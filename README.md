# Tienda

This is a ionic multiplatform store PWA, using: angular 17.2.1 and firebase.

## Installationfg

To install dependencies:
```bash
cd tienda
npm install
```
## Install ionic and angular
```bash
npm install -g @ionic/cli
npm install -g @angular/cli@17.2.1
```

## Run the App:
```bash
ionic serve
```
